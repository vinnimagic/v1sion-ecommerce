# V1SION E-commerce

![Logo](./public/images/logo.png)

E-commerce moderno com carrinho de compras, histórico de pedidos e modo escuro.

## Funcionalidades

- 🛒 Carrinho de compras interativo
- 🌓 Modo claro/escuro com persistência
- 📦 Histórico de pedidos
- 🔍 Filtro de produtos
- 📱 Design responsivo

## Tecnologias

- React 18
- React Router 6
- Context API
- CSS Modules
- LocalStorage
